# Caixa App

Sistema de controle financeiro desenvolvido com React + Laravel.

## 🚀 Tecnologias

### Frontend
- React
- TypeScript
- TanStack Router
- TailwindCSS
- Shadcn/UI

### Backend
- Laravel
- Sanctum
- MySQL

## 🔐 Autenticação

O projeto utiliza autenticação SPA com Laravel Sanctum baseada em sessão/cookies HTTP-only.

## 📦 Funcionalidades

- Login e logout
- Dashboard financeiro
- Controle de entradas e saídas
- Relatório semanal
- Categorias
- Rotas protegidas
- API REST
